# Cursor 团队日报 — 成员部署指南（小白版）

> **2026-06 更新：** 同事请优先使用 [Skill安装与使用指南](Skill安装与使用指南.md) 或 Frank 发的「三件装」zip，**无需 Git**。  
> 下文含 Git clone 步骤，仅供主管或需要完整仓库的人参考。

> 发给每位同事按步骤操作即可。全程约 15 分钟，**只需配置一次**。

---

## 这套东西是干什么的？

你每天用 Cursor 写文档、改代码、问 AI，系统会在**下班前自动**帮你总结「今天用 Cursor 做了什么」，写入数据库，主管可以查看。

**你不需要：**
- 单独申请账号密码
- 每天手工写日报
- 懂 Git、数据库

**你需要：**
- 电脑已安装 **Cursor** 和 **Python 3.10+**
- 按下面步骤配置一次

---

## 第一步：下载代码

打开 PowerShell，执行：

```powershell
cd D:\
git clone https://github.com/Frankie-Foo/cursor-team-daily-report.git
cd cursor-team-daily-report
```

---

## 第二步：一键安装

```powershell
powershell -ExecutionPolicy Bypass -File scripts/setup.ps1
```

---

## 第三步：填写你的个人信息

### 3.1 复制配置文件

```powershell
copy config\user.example.json config\user.json
copy .env.example .env
```

### 3.2 编辑 `config\user.json`

用 Cursor 或记事本打开，改成**你自己的信息**：

```json
{
  "username": "April",
  "timezone": "Asia/Shanghai",
  "cursor_workspace": "D:/你平时打开的那个Cursor项目文件夹"
}
```

**重要说明：**

| 字段 | 填什么 |
|------|--------|
| `username` | 你的英文名，必须和下表完全一致 |
| `cursor_workspace` | 你**每天实际干活**时 Cursor 打开的项目路径，不是本日报仓库 |

**全员 username 对照表（大小写一致）：**

### 3.3 编辑 `.env`

## 第六步：设置每天自动运行（推荐）

在 Cursor 里打开 **Automations**，新建定时任务：

| 设置项 | 填什么 |
|--------|--------|
| 名称 | Cursor 团队日报 |
| 时间 | 工作日 17:30 |
| Cron | `30 17 * * 1-5` |
| Skill | `cursor-daily-report` |

**Automation 指令（复制粘贴）：**

```text
运行 cursor-daily-report skill，发布今日日报：
1. 读取 config/user.json 的 username 和 cursor_workspace
2. 执行 python scripts/publish_daily.py --date today --git-push
3. 若今天没有 Cursor 会话，仍发布空日报
```

配好后，**以后不用管**，每天自动汇总。

## 常见问题

### Q：报「未找到 agent-transcripts」？

`config/user.json` 里的 `cursor_workspace` 填错了。填你**日常打开的业务项目**，例如 `D:/经销商PDCA`，不是 `cursor-team-daily-report` 这个仓库。

### Q：报「未配置用户名」？

检查 `config/user.json` 是否存在，且 `username` 与上表一致。

### Q：报数据库连接失败？

`.env` 里 `DB_PASSWORD` 是否正确？找 Frank 确认。

### Q：需要每天手动操作吗？

配好 Automation 后**不用**。偶尔也可手动：

```powershell
python scripts/publish_daily.py --date today --git-push
```

### Q：主管能看到我什么？

- **普通成员**：只能看自己
- **小组长**：看自己组内的人（如 Lina 看 Lina + Viki）
- **总监**：看全员

不需要额外登录，系统根据你 `username` 自动区分。
