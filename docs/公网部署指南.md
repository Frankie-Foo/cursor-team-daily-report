# 公网 POST 部署指南（必做）

同事在家/外网 **无法访问** `10.100.0.176`（内网 IP）。必须把 API 暴露到 **公网 HTTPS 地址**，所有人 `.env` 里的 `REPORT_API_URL` 填这个地址。

---

## 推荐：Cloudflare Tunnel（不用找 IT 开端口）

API 仍跑在内网机器（如 `10.100.0.176`），由 `cloudflared` 把 `localhost:8080` 映射到公网 HTTPS。

### A. 快速验证（临时 URL，重启会变）

在 **跑 API 的那台机器** 上：

```powershell
cd D:\cursor-team-daily-report

# 终端 1：启动 API
powershell -File scripts/run_api.ps1

# 终端 2：临时公网隧道
powershell -File scripts/run_cloudflare_quick_tunnel.ps1
```

终端 2 会打印类似：

```text
https://xxxx.trycloudflare.com
```

把 `.env` 和同事包里的地址改成：

```env
REPORT_API_URL=https://xxxx.trycloudflare.com
```

然后重新打包分发：

```powershell
powershell -File scripts/build_colleague_package.ps1
python scripts/export_member_credentials.py
```

> 临时 URL **每次重启 tunnel 都会变**，只适合验证；正式环境用下面 B。

### B. 正式固定域名（推荐）

前提：有一个域名在 Cloudflare 管理（如 `vertu.cn` 子域名 `cursor-reports.vertu.cn`）。

1. 安装 cloudflared：`winget install Cloudflare.cloudflared`
2. 登录：`cloudflared tunnel login`
3. 创建隧道：`cloudflared tunnel create cursor-team-reports`
4. 复制 `config/cloudflare-tunnel.example.yml` 为 `config/cloudflare-tunnel.yml`，填入 tunnel id 和域名
5. 配置 DNS：`cloudflared tunnel route dns cursor-team-reports cursor-reports.你的域名.com`
6. 写入 `.env`：

```env
REPORT_API_URL=https://cursor-reports.你的域名.com
CLOUDFLARE_TUNNEL_CONFIG=D:/cursor-team-daily-report/config/cloudflare-tunnel.yml
```

7. 一条命令启动 API + 固定隧道：

```powershell
powershell -File scripts/run_api_public.ps1
```

8. 外网验证：

```powershell
curl https://cursor-reports.你的域名.com/api/v1/health
```

9. 注册 Windows 计划任务常驻（开机自启）：

```powershell
powershell -File scripts/install_api_public_task.ps1
```

## 备选：公司 Nginx / 公网服务器反代

若 IT 能在公网服务器（如 `admin.vertu.cn` 同机）加反代：

```nginx
location /cursor-reports/ {
    proxy_pass http://10.100.0.176:8080/;
    proxy_set_header Host $host;
    proxy_set_header X-Forwarded-Proto $scheme;
}
```

则：

```env
REPORT_API_URL=https://admin.vertu.cn/cursor-reports
```

## 同事侧配置

每人 `.env` **只需改这一行**（Frank 统一通知新公网地址）：

```env
REPORT_API_URL=https://你的公网地址
REPORT_API_TOKEN=Frank私发的个人Token
```

连通性自检：

```powershell
powershell -File scripts/test_api_connection.ps1
```

---

## 安全说明

- 已启用 **Bearer Token 鉴权**，公网暴露可接受
- 务必使用 **HTTPS**（Cloudflare Tunnel 自动提供）
- `config/api_tokens.json` 勿提交 Git、勿群发
