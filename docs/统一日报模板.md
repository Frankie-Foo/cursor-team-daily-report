# 统一日报模板

每日自动聚合 **Vertu 业务日报 + Vemory 会议 + Cursor 工作**，POST 到 API。

---

## 结构

```
统一日报
├── OKR 进度（来自 Vertu）
├── 一、Vertu 今日工作（任务/进度/工时/状态）
├── 二、Vemory 当日会议（摘要/待办）
├── 三、Cursor 当日工作（会话摘要）
├── 明日计划
└── 重点事项（自动提炼）
```

JSON 字段 `sections` 存三大板块原始数据，`daily_summary` 存 Markdown 全文。

---

## 固定每日 POST

**标准命令（唯一）：**

```powershell
powershell -File run_daily.ps1
```

**Windows 定时（工作日 17:30）：**

```powershell
powershell -File install_daily_task.ps1
```
