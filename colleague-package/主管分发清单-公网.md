| 文件 | 路径 |

---

## 2. 公网 API 地址（全员相同）

```
REPORT_API_URL=见 config/member_credentials.md 顶部
```

---

## 3. 私发（每人不同）

打开 **`config/member_credentials.md`**，按人复制「逐人私发话术」块，**微信私发**，勿群发 Token。

或打开 **`config/member_credentials.json`** 给脚本/自动化用。

---

## 4. 同事安装要点

1. 解压 zip → Cursor `@` 文件夹
2. 复制 `.env.example` → `.env`，填入 **公网 REPORT_API_URL** + **个人 Token**
3. 配置 `config/user.json`（username / odoo_user_id / cursor_workspace）
4. `powershell -File scripts/test_api_connection.ps1` 必须 **OK**
5. `powershell -File install_daily_task.ps1` 注册工作日 17:30

---

## 5. 主管运维

```powershell
# 看谁交了
python scripts/query_team.py --viewer Frank --status

# 公网服务栈（API + Tunnel）
powershell -File scripts/start_production_stack.ps1

# 换固定域名后重新打包
python scripts/set_production_url.py --public-url https://新域名
powershell -File scripts/build_colleague_package.ps1
python scripts/export_member_credentials.py
```

正式隧道配置见 **`docs/公网部署-正式版.md`**。
